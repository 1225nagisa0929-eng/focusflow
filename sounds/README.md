# Sound Files for FocusFlow

Place the following sound files in this directory:

## Required Files

1. **focus-start.mp3** - Gentle chime when focus session starts
   - Recommended: Soft bell or chime (2-3 seconds)
   - Volume: Medium-low

2. **break-start.mp3** - Calming tone when break begins
   - Recommended: Soft water/nature sound (2-3 seconds)
   - Volume: Low

3. **complete.mp3** - Celebration sound for completed session
   - Recommended: Pleasant completion tone (3-4 seconds)
   - Volume: Medium

## ADHD-Friendly Sound Guidelines

- Avoid harsh or startling tones
- Keep sounds short (2-4 seconds)
- Use soft, rounded frequencies
- Avoid high-pitched or sharp sounds
- Test with ADHD users if possible
